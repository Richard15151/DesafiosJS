Desafios de javaScript

Desafio 1 - Grito

Para este desafio, o script seleciona o botão usando getElementById. Em seguida, encadeia diretamente o método addEventListener para escutar por eventos de 'click'. Uma arrow function é utilizada como callback para, de forma concisa, disparar um alert() com uma mensagem específica quando o botão é clicado.
