Desafio 8 - Menu interativo

  • HTML ✔
  • CSS ✔
  • JavaScript ✔

Esta solução usa event delegation no ul principal (#menuprincipal). Ao clicar, closest() identifica o li clicado (com data-tipo). O script gerencia o estado do item ativo atual. Funções auxiliares alternam os estilos visuais do item ativo (negrito, ícone) e exibem o conteúdo correspondente (armazenado em um objeto, acessado via data-tipo) na div (#caixa), tornando-a visível.
