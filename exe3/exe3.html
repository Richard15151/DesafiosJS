Desafio 3 Manipulando com DOM

Este script primeiro obtém referências ao título (h1) e ao botão via getElementById. Um addEventListener no botão executa uma função que verifica o innerHTML atual do título usando uma estrutura if/else. Com base no texto existente, ele atualiza o innerHTML para a string alternativa, alternando o conteúdo do título a cada clique.
